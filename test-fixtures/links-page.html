JavaScript Guide

This guide covers the fundamentals of JavaScript programming. Follow the chapters below to learn step by step.

Chapters

  1. Introduction - Getting started with JavaScript basics
  2. Grammar and types - Variables, data types, and literals
  3. Control flow and error handling - Conditionals, loops, and exceptions
  4. Loops and iteration - For, while, and do-while loops
  5. Functions - Defining and calling functions
  6. Expressions and operators - Arithmetic, comparison, and logical operators
  7. Numbers and dates - Working with numeric data and date objects

About this guide

This guide assumes you have a basic understanding of HTML and CSS. Each chapter builds on the previous one, so it is recommended to read them in order.
